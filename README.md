# React accordion - Technology Group (test-task)

## Stack

### Front:

- React
- TypeScript
- Styled components
- Framer Motion

### Back:

- Express

## How to install?

1. `cd ./front`
2. `npm install` (client)
3. `cd ../back`
4. `npm install` (server)

## How to use?

### First terminal

1. `cd ./front`
2. `npm start`

### Second terminal

2. `cd ./server`
3. `npm start`
